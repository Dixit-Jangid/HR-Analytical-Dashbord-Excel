# HR-Analytical-Dashbord-Excel

->An interactive HR Analytics Dashboard built in Microsoft Excel to convert raw data into meaningful insights and support data-driven decisions.

# Tools Used
- Microsoft Excel  
- Data Cleaning  
- Pivot Tables & Charts  
- Slicers & KPI Cards  
- Data Visualization  

# Key Insights
- Revenue, Orders, Quantity & Ratings overview  
- Weekly sales trends  
- Customer behavior analysis  
- Product popularity by gender  
- Shipping time & customer satisfaction  


-> This project helped me understand data cleaning, dashboard design, and storytelling with data.
